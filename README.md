# OpenClaw Founder Assistant

AI chief of staff built with OpenClaw. Runs on Discord.
Handles morning briefings, email drafting, team Q&A, and fallback responses.

## Watch the demo
[4-minute live demo — https://www.loom.com/share/ef05e56d533c4cd2999f7b7d40259d7c]

## What it does
- Morning briefing with company context and priorities
- Email drafting in the founder's voice
- Team Q&A from a custom knowledge base
- Graceful fallback for unknown questions
